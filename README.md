# 비밀번호 난수 생성기

여러 개의 비밀번호를 한 번에 생성할 수 있는 간단한 웹 앱입니다.

지원 기능:

- 비밀번호 길이 조절
- 생성 개수 선택
- 숫자, 소문자, 대문자, 기호 포함 여부 설정
- 유사 문자 제외
- 애매한 문자 제외
- 같은 요청 안에서 중복 없이 생성
- 결과를 줄바꿈 텍스트로 표시

## 로컬 실행

```bash
python3 /Users/indeepai/Documents/codex/app.py
```

브라우저에서 `http://127.0.0.1:8000` 으로 접속하면 됩니다.

## 같은 와이파이에서 공유

현재 서버는 `0.0.0.0:8000` 으로 열리도록 되어 있어서, 같은 와이파이의 다른 기기도 내부 IP 주소로 접속할 수 있습니다.

예시:

```text
http://192.168.x.x:8000
```

## Render 무료 배포

이 프로젝트는 Render 무료 웹 서비스 기준으로 배포 가능하게 준비되어 있습니다.

필요 파일:

- `app.py`
- `render.yaml`
- `.python-version`
- `requirements.txt`

배포 순서:

1. 이 폴더를 GitHub 저장소로 올립니다.
2. Render에서 새 `Blueprint` 또는 `Web Service`를 생성합니다.
3. GitHub 저장소를 연결합니다.
4. Render가 `render.yaml`을 읽어 배포합니다.

배포 후에는 Render가 제공하는 URL로 접속하면 됩니다.
